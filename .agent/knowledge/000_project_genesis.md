# 🚀 Chào mừng AI đến với dự án: ReaperAI_Karaoke

> **LỆNH TỐI THƯỢNG CHO ANTIGRAVITY & CÁC AI KHÁC:**
> Khi bạn đọc được file này ở đầu Session, hãy ngầm hiểu đây là bối cảnh cốt lõi của dự án. 
> Đừng hỏi Sếp những câu hỏi cơ bản về "Dự án này stack là gì". Hãy tự dùng công cụ `vfs_assistant (MCP)` hoặc `list_dir` để quét lướt qua thư mục `src/` hoặc các file config (như package.json) để tự rút ra cấu trúc.

## Mục tiêu
- Giúp AI (đặc biệt là Antigravity) có Context ngay từ tin nhắn đầu tiên, mô phỏng Zero-Shot Codebase Mapping.
- Định hướng AI đọc `.agent/GEMINI.md` để tuân thủ rule hệ thống.

## 🛠️ Trạng thái Hệ thống
- Tên dự án: **ReaperAI_Karaoke**
- Trạng thái: AG-Kit vừa được khởi tạo thành công.
- Các Pack kỹ năng đã cài: `core,web,python,bot`

## 🎯 Hành động tiếp theo của AI
1. Quét `package.json` hoặc các file root config để update Tech Stack vào nhận thức.
2. Phân tích cấu trúc thư mục để hiểu Architecture.
3. Sẵn sàng nhận Lệnh tác chiến từ Sếp.
